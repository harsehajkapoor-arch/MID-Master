Auto-Fill a Raw Manifest
How it works: Upload any raw manifest file. The tool will scan every row, find blank itemManufacturerMID or itemManufacturerName cells, and fill them using the database — only if it finds an exact match. If a row has a MID conflict across manifests, it flags it and does NOT fill automatically. Anything it cannot resolve is clearly marked so you can review manually. The final filled file is then available to download.

MID Conflict Detector
A conflict means the same manufacturer name appears with different MIDs across different manifest files. This could indicate a data entry error, name variation, or a genuine manufacturer change. These rows are flagged and will NOT be auto-filled — you must resolve them manually.

2. Generate a new token with repo scope checked
3. Paste it below. It is saved only in this browser's localStorage — never sent anywhere except GitHub's API.
